🔍 核心问题与解决方案
══════════════════════════════════════════════════════════════════════════

问题: 400 User location is not supported for the API use.

原因: Google Gemini API 检测到您的服务器位置在非支持地区

解决方案 (按推荐顺序):

1️⃣  使用免费代理
   • 访问: https://free-proxy-list.net/
   • 获取代理 IP 和端口
   • 设置: export HTTP_PROXY=http://IP:PORT
   • 设置: export HTTPS_PROXY=https://IP:PORT

2️⃣  使用 VPN
   • OpenVPN, WireGuard 或商业 VPN
   • 连接到支持的地区 (美国、欧洲等)

3️⃣  使用付费代理服务
   • Bright Data, Smartproxy, Oxylabs 等
   • 更稳定的连接质量

4️⃣  替换 API 提供商
   • OpenAI (GPT)
   • Azure OpenAI
   • Anthropic (Claude)
